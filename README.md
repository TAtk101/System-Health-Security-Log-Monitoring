# System Health & Security Log Monitoring

## Project Overview
This project is an **Endpoint Monitoring and Security Log Collection tool** built with PowerShell.  
It collects key system metrics, including CPU usage, available memory, free disk space, and system uptime.  
The script automatically exports a **CSV report** to the Desktop for easy analysis.

This project demonstrates skills in:
- IT Operations Monitoring
- Endpoint Health Assessment
- PowerShell Scripting & Automation
- Data Collection & CSV Reporting
- OneDrive/Windows Environment Awareness

---

## Features
- Collects CPU usage, memory, disk space, and system uptime
- Generates a structured CSV report
- OneDrive-safe file path detection
- Safe null-checks to prevent errors
- Ready for automation with Task Scheduler
